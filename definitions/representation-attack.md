# Representation Attacks

## Canonical Definition

Representation Attacks are attacks that manipulate how AI systems internally represent reality rather than directly attacking the final output layer.

These attacks target:
- embeddings
- contextual representations
- entity mappings
- semantic interpretation
- memory systems
- retrieval layers
- prompts
- latent structures
- knowledge associations

The goal is to distort how the system understands reality.

---

# Why It Matters

AI systems increasingly depend on representations rather than deterministic rules.

This creates new attack surfaces:
- adversarial examples
- prompt injection
- context poisoning
- embedding manipulation
- retrieval corruption
- entity confusion
- semantic drift

---

# Key Insight

A representation attack may appear harmless externally while fundamentally changing internal reasoning.
